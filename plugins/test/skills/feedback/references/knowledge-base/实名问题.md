## 🔍 问题分类

### 1. TapTap 账号实名问题

**特征**：
- 用户在 TapTap APP 内提交实名
- 实名信息提交失败或报错
- 实名状态显示异常

**判断标准**：
- ✅ 是 TapTap 的问题
- ✅ 业务线：TapSDK（飞书系统枚举值）
- ✅ 对接人：赵俊杰

**常见原因**：
1. 实名信息格式不正确
2. 实名服务异常
3. 中宣部接口返回失败
4. 证件号码已被占用

### 2. 游戏内实名认证问题

**特征**：
- 游戏内调用 TapSDK 实名认证
- TapTap 账号已实名，但游戏内认证失败
- 快速实名认证失败，转手动实名

**判断标准**：
- ⚠️ 需要区分是游戏侧问题还是 SDK 问题
- ✅ 业务线：TapSDK（飞书系统枚举值）
- ✅ 对接人：赵俊杰

**常见原因**：
1. 游戏侧 SDK 接入问题
2. 实名信息同步延迟
3. 游戏侧实名接口调用错误
4. SDK 版本过旧

## 💡 常见问题

### Q1: TapTap 已实名，游戏内还提示未实名？

**可能原因**：
1. 实名信息同步延迟（等待5-10分钟）
2. 游戏侧缓存问题（清除缓存重试）
3. SDK 版本过旧（需要游戏方升级）
4. 游戏侧接入问题（联系游戏开发者）

**处理建议**：
- 先让用户等待几分钟后重试
- 如果持续失败，创建 Bug 联系赵俊杰

### Q2: 实名认证提示"错误的请求"？

**可能原因**：
1. 实名信息格式不正确
2. 证件号码已被其他账号使用
3. 实名服务异常
4. 中宣部接口返回失败

**处理建议**：
- 检查用户输入的实名信息是否正确
- 后台查看是否有重复证件号
- 创建 Bug 让赵俊杰排查后端问题

### Q3: 快速实名认证失败，转手动实名？

**可能原因**：
1. 中宣部返回认证失败
2. 用户信息与中宣部数据不匹配
3. 网络问题导致认证超时

**处理建议**：
- 引导用户使用手动实名
- 创建 Bug 让赵俊杰确认具体原因
- 如果是中宣部问题，可能需要用户联系相关部门
