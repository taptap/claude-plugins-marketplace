## 🔍 使用方式

### 在分析 Feedback 时

当 AI 分析 #taptap-feedback 频道的消息时，会：

1. **识别问题类型**（如：实名问题、SDK 登录问题等）
2. **查找知识库**中的相关历史案例
3. **参考历史处理经验**进行问题分析
4. **推荐对应负责人**（从历史案例中的对接人）

### 查询示例

**场景 1：SDK 登录问题**
```markdown
请查找知识库中关于 "TapTap 登录提示 405" 的历史案例
```

AI 会在 `TapSDK问题.md` 中查找相关案例，并给出：
- 历史案例描述
- 可能原因
- 对接人信息
- Slack 历史讨论链接

**场景 2：实名认证问题**
```markdown
请查找知识库中关于 "快速实名认证失败" 的案例
```

AI 会在 `实名问题.md` 中查找并给出处理建议。
