## 🔍 问题识别

### 活动配置问题特征

- 活动页面显示异常
- 活动链接失效（404）
- 活动规则描述错误
- 活动时间显示不正确
- 活动奖品配置错误

**判断标准**：
- ✅ 不是技术故障
- ✅ 需要调整活动后台配置
- ✅ 通常是紧急问题（影响活动进行）

### 游戏信息配置问题特征

- 游戏介绍信息错误
- 开发者信息不正确
- 官网链接失效或错误
- 游戏标签分类错误
- 游戏封面/截图问题

**判断标准**：
- ✅ 不是技术故障
- ✅ 需要在 DC 后台调整
- ⚠️ 可能涉及合规问题（如链接内容不当）

### 编辑推荐语问题特征

- 推荐文案中有错别字
- 推荐文案描述不准确
- 推荐文案格式问题

**判断标准**：
- ✅ 不是技术故障
- ✅ 需要内容编辑调整
- 🟡 优先级通常较低

## 📝 处理流程

### 1. 识别问题类型

根据用户反馈判断属于哪种配置问题。

### 2. 不创建 Bug，直接联系对接人

配置问题通常**不需要创建飞书 Bug**，直接：
- 在 Slack 中 @对应对接人
- 提供详细信息（游戏名、链接、截图等）
- 说明问题紧急程度

### 3. Slack 回复模板

**活动配置问题**：
```markdown
已转给活动团队处理

@颜成宇 @张娜 请帮忙看下活动配置问题：
- 游戏/活动：[名称]
- 问题描述：[详细说明]
- 用户反馈：[工单链接]

预计处理时间：[根据紧急程度]
```

**游戏信息配置**：
```markdown
已转给内容团队处理

@童洁 @刘慧丽 请帮忙看下游戏信息配置：
- 游戏：[名称]
- 问题：[详细说明]
- 链接：[相关链接]

预计处理时间：1-2个工作日
```

**编辑推荐语**：
```markdown
已转给编辑团队

@康骥 请帮忙看下推荐文案：
- 游戏：[名称]
- 位置：[推荐页/其他]
- 问题：[错别字或描述问题]
```

## 💡 常见问题

### Q1: 如何判断是配置问题还是技术问题？

**配置问题特征**：
- 内容显示错误（文字、图片、链接）
- 后台设置需要调整
- 人工可以修改

**技术问题特征**：
- 功能无法使用
- 系统报错
- 需要代码修改

### Q2: 活动配置问题很紧急怎么办？

1. 立即在 Slack #taptap-operations 频道通知
2. @活动负责人：颜成宇、张娜
3. 说明紧急程度和影响范围
4. 如果是 P0 级别，可能需要拉专项群

### Q3: 游戏信息涉及合规问题怎么办？

如果游戏信息（如官网链接）涉及违规内容：
1. 立即通知内容团队：@童洁 @刘慧丽
2. 如果情况严重，同时通知风控团队
3. 可能需要紧急下架或隐藏链接
