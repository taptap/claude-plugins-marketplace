## 📚 问题分类与案例

### 1. 登录问题

#### 案例 1.1：Android 7.1 以下登录失败

**问题描述**：
TapTap 登录授权后会弹出错误代码，且设备系统在 Android 7.1 以下。

**根本原因**：
**游戏兼容性问题** - 游戏使用了内建账户的默认域名，但默认域名不支持 Android 7.1 以下版本。

**解决方案**：
需要开发者自行更换为自定义域名。

**历史案例**：
- 游戏：《放置文明》、《钓鱼佬净化海洋》、《迷雾城堡》
- 用户 ID：590733602
- [Slack 讨论](https://xindong.slack.com/archives/C01EB1KAE4B/p1729565334711929)

---

#### 案例 1.2：登录提示"错误的请求 405"

**问题描述**：
游戏内通过 TapTap 登录，提示"错误的请求 405"。

**根本原因**：
**用户请求网络环境问题** - 请求 IP 是 IDC IP，大概率会被拦截，比如用了云游戏、挂了代理、使用腾讯云/阿里云等云 IP。

**解决方案**：
可更换 IP 重试。

**历史案例**：
- 游戏：《大侠立志传》
- Tap ID：9202091
- [Slack 讨论](https://xindong.slack.com/archives/C01EB1KAE4B/p1726813755270019)

---

#### 案例 1.3：人机验证提示"操作过于频繁"

**问题描述**：
使用 Tap 登录，人机验证提示"您的操作过于频繁，请稍后再试"。

**根本原因**：
**外部依赖** - 人机验证供应商的风控策略。

**解决方案**：
需更换网络或等待半小时后重试。

**历史案例**：
- 游戏：代号：旅程

**解决方案**：
需要游戏开发者检查包名和签名配置。

#### 案例 1.5：提示"当前游戏还未开放测试计划"

**问题描述**：
游戏内通过 TapTap 登录，提示"当前游戏还未开放测试计划呢~"。

**根本原因**：
**游戏在 DC 后台配置问题** - 创建了测试计划且开启了"仅有测试资格可登录游戏"。

**解决方案**：
- 检查 DC 后台测试计划配置
- 确认用户是否有测试资格
- 或关闭测试资格限制

### 2. 合规认证问题

#### 案例 2.1：已实名成年人，游戏提示未成年

**问题描述**：
Tap 已实名成年人用户，进入游戏仍提示未成年。

**根本原因**：
**已知缺陷** - 游戏内同一时间多次调用 startup 接口导致实际获得错误的回调结果。

**解决方案**：
1. 游戏侧对调用 startup 接口做防抖动处理
2. 升级 TapSDK 至 3.29.5 / 4.5.0 及以上版本

**历史案例**：
- 游戏：《反转21克》
- 用户 ID：608076385

### 3. 正版验证问题

#### 案例 3.1：登录后自动跳转详情页，不进入游戏

**问题描述**：
游戏内使用 TapTap 登录后，每次自动跳转 TapTap 客户端内游戏详情页，不进入游戏。

**根本原因**：
**游戏侧付费验证接入问题** - 检查是否购买本体。

**解决方案**：
- 确认用户是否购买游戏
- 检查游戏付费验证接口接入
- 验证订单状态

**历史案例**：
- 游戏：《阿尔托的冒险》

### 5. 更新唤起问题

#### 案例 5.1：运行游戏自动打开 TapTap 详情页

**问题描述**：
运行游戏时会自动打开 TapTap 客户端内游戏详情页。

**根本原因**：
**游戏侧更新唤起接入问题** - 需自行维护版本校验接口。

**解决方案**：
游戏开发者需要：
1. 正确实现版本校验逻辑
2. 避免每次启动都触发更新检查
3. 只在需要更新时才唤起 TapTap

**历史案例**：

## 🔍 快速判断流程

### 1. 确认是否是 SDK 问题

```
是游戏调用 TapSDK 功能时出现异常吗？
└─ 是 → 继续
└─ 否 → 不是 SDK 问题，可能是商店/主站问题

截图或日志中是否含 "TapTap" 标识？
└─ 是 → 继续
└─ 否 → 需要进一步确认

问题类型是什么？
├─ 登录授权 → SDK 登录问题
├─ 实名认证 → SDK 合规认证问题
├─ 付费验证 → SDK 正版验证问题
├─ 成就同步 → SDK 成就问题
└─ 更新唤起 → SDK 更新唤起问题
```

### 2. 确认对接人

```
问题类型确认后：
├─ 登录/更新唤起 → 潘扬（TL）
├─ 实名/合规认证 → 赵俊杰
├─ 成就同步 → 宁佳晨
└─ 游戏侧接入问题 → 王浩/余文健
```
